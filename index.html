Surferpedia

Play web application providing biographical information about surfers.

Download this project as a .zip file Download this project as a tar.gz file

Overview A Play application that provides an encyclopedia of surfers.

Installation Download the zip file, unzip, cd into the directory, invoke the Play console, and execute the run command to see the application at http://localhost:9000.

Usage Use the navbar to find interior pages with bio information on surfers.
